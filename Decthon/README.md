- **Dark Mode** - Beautiful dark theme by default

## Quick Start

### 1. Backend Setup

```bash
cd backend
cp .env.example .env
# Add your OpenAI API key to .env
pip install -r requirements.txt
python main.py
```

### 2. Open Frontend

Open `structure.html` in your browser, or:

```bash
open structure.html
```
